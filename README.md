# Minimalist Modern Email Signature

A clean, professional, and responsive HTML email signature template. This design features a modern glowing profile picture effect and integrated social media icons, perfect for personal branding.

## Key Features
- **Modern Aesthetic:** Sleek layout with a soft-glow profile picture border.
- **Fully Responsive:** Optimized for desktop, tablet, and mobile displays.
- **Social Integration:** Includes clickable icons for Facebook, Instagram, Twitter, and LinkedIn.
- **Cross-Platform Compatibility:** Table-based structure to ensure it looks great in Gmail, Outlook, and Apple Mail.
- **Easy to Edit:** Clean, well-commented HTML/CSS code.

## Installation
1. Clone this repository to your local machine.
2. Open `index.html` in your code editor.
3. Update the text details (Name, Title, Contact) and image source.
4. Update the `href` links for social media icons.
5. Copy the final code into your email client's signature settings.

## Built With
- HTML5
- CSS3 (Flexbox & Box-Shadow effects)
